# NumPy Project

## Overview
This project demonstrates the use of NumPy for data analysis through a sports academy scouting scenario. It focuses on array manipulation, broadcasting, structured arrays, filtering, sorting, and selecting the best players based on a weighted performance score.

## Project Objectives
- Explore and inspect NumPy arrays
- Clean and preprocess data
- Apply broadcasting with weighted calculations
- Build structured and record arrays
- Filter and sort data
- Select the best players based on predefined rules
- Generate a final performance report

## Technologies Used
- Python
- NumPy

## Skills Demonstrated
- Array indexing and slicing
- Array copying
- Broadcasting
- `np.clip()`
- Structured Arrays
- Record Arrays
- Filtering
- Sorting
- Statistical calculations
