# Titanic Cleaning

## Overview
This project focuses on cleaning and preprocessing the Titanic dataset using Python and Pandas. It includes handling missing values, filtering data, creating new features, and performing basic exploratory data analysis (EDA).

## Dataset
- Titanic Dataset (CSV)

## Tools & Libraries
- Python
- Pandas
- NumPy

## Project Tasks
- Data inspection
- Selecting and filtering data
- Handling missing values
- Creating new features
- Grouping and aggregation
- Pivot tables
- Extracting passenger titles
- Basic data analysis
