# Titanic Explore

## Overview
This project focuses on cleaning and exploring the Titanic dataset using Python and Pandas. The dataset was inspected, analyzed, and checked for data quality issues before further analysis.

## Dataset
- Titanic Dataset (887 rows × 8 columns)

## Tools & Libraries
- Python
- Pandas

## Project Steps
- Loaded the dataset using Pandas.
- Displayed the first five rows.
- Inspected dataset information and data types.
- Checked the dataset shape.
- Generated descriptive statistics.
- Checked for duplicate records.
- Checked for missing values.

## Results
- Dataset Size: **887 rows × 8 columns**
- Missing Values: **None**
- Duplicate Rows: **0**
